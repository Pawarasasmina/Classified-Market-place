# Project Plan

## Goal
Set up a clean monorepo foundation for a 2-developer MVP Classified Marketplace Platform.

## Apps
- `apps/web`: Next.js web app
- `apps/mobile`: Expo React Native app
- `apps/api`: NestJS backend API
- `packages/shared`: shared types/constants

## Early Milestones
1. Auth baseline
2. Listing CRUD
3. Search/filter
4. Chat + notifications
5. Admin moderation basics
